## Getting Started

To get a local copy up and running follow these simple steps:

### Prerequisites

  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repository.

   ```sh
   git clone https://github.com/santiagomorelle/auth-project.git
   ```
   
2. Install NPM packages.

   ```sh
   npm install
   ```
   
### Usage

- Run the app!

   ```sh
   npm run dev
   ```
